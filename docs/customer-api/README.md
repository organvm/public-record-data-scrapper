# UCC-MCA Intelligence Platform Customer API

Welcome to the official Customer API documentation for the UCC-MCA Intelligence Platform.

This API allows you to programmatically access Merchant Cash Advance (MCA) lead prospects derived from Uniform Commercial Code (UCC) filings, manage funded portfolios, track competitors, and enrich prospects with data from 60+ sources.

## Base URLs

* **Development:** `http://localhost:3000/api`
* **Production:** `https://api.your-domain.com/api`

## Getting Started

1. **Authentication:** Obtain a Bearer token to authorize your requests. See [Authentication](AUTHENTICATION.md).
2. **Explore Endpoints:** Browse the available RESTful endpoints for [Prospects, Competitors, Portfolio, and Enrichment](ENDPOINTS.md).
3. **Usage Examples:** Jumpstart your integration with code examples in cURL, JavaScript, and Python in the [Examples](EXAMPLES.md) guide.
4. **Limits and Errors:** Understand [Rate Limits and Error Handling](ERRORS_AND_LIMITS.md) to build robust integrations.
